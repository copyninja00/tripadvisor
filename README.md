TripAdvisor Clone 🌍

A responsive frontend clone of the TripAdvisor homepage built using HTML and CSS.
This project recreates the layout and user interface of TripAdvisor to practice modern web design, responsive layouts, and component structuring.

✨ Features
Responsive navigation bar
Hero search section
Category cards with images
Experience and suggestion sections
Promotional banners
Footer with multiple menus
Responsive layout using media queries
🛠️ Technologies Used
HTML5
CSS3
Flexbox
Media Queries
Google Fonts
📱 Responsive Design

The project includes responsive breakpoints to ensure the layout adapts across different screen sizes including desktop, tablet, and mobile devices.

🎯 Purpose

This project was built to improve frontend development skills by replicating a real-world website layout and practicing responsive UI design.
